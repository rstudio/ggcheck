# ggcheck: Inspect 'ggplot2' Plots for Automated Grading in Learning Exercises

'ggcheck' provides functions that inspect 'ggplot2' objects to make it
easier for teachers to check that student plots meet expectations.
Designed primarily for automated grading via 'gradethis' in interactive
'learnr' tutorials.
